Welcome to MailWarden

MailWarden is an AI-powered spam filter that runs entirely on your Mac. It connects to your own email account, uses your own Anthropic API key to classify incoming messages, and moves anything it flags as spam into your junk folder. It learns from emails you forward it and is controlled entirely by you.

This installer will:

  • Install MailWarden.app into /Applications/.
  • On first launch, set up ~/MailWarden/ with the filter code and the background tasks that run it on an interval you choose (default 15 minutes, adjustable from 5 to 360 minutes).
  • Walk you through a one-time Setup Assistant that asks for your Anthropic API key, your email credentials, and a few preferences.

Install on one Mac per email account. MailWarden should run on only one Mac per set of email accounts. If you install it on a second Mac that also checks the same email, you will pay for every spam classification twice and your inbox will get duplicate confirmation replies. Install on multiple Macs only when each Mac monitors a different, non-overlapping set of accounts.

What you will need:

  • An Anthropic API key (starts with sk-ant-). Get one at console.anthropic.com.
  • Your email credentials. Gmail and AOL require an App Password, not your regular account password.
  • About five minutes.
