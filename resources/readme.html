About this installer

MailWarden installs two things:

1. /Applications/MailWarden.app

The graphical interface. Launches the Setup Assistant on first run and the Dashboard on every subsequent run. You can remove it by dragging MailWarden.app to the Trash.

2. ~/MailWarden/

The filter runtime. Contains the code that runs on an interval you choose (default 15 minutes, adjustable from 5 to 360 minutes), the learned spam signals, your whitelist and blacklist, your API usage history, and logs. This folder is created on your first launch of MailWarden.app — nothing is written here during .pkg installation.

Background tasks

MailWarden registers three background agents through macOS (Apple's SMAppService) from inside the app bundle the first time you launch it. They appear in System Settings → General → Login Items, where you approve MailWarden:

  • com.mailwarden.filter — runs the spam filter on your chosen interval (default 15 minutes, adjustable from 5 to 360 minutes).
  • com.mailwarden.report — sends you a daily summary email at the time you choose (default 8:00 AM).
  • com.mailwarden.menubar — starts the menu bar icon at login, if enabled.

Anthropic API usage

MailWarden uses your own Anthropic account to classify emails. API charges are billed directly to Anthropic, not to STR Solutions. Typical usage is pennies per day. You can check usage any time from the Dashboard.

Upgrades

If you install MailWarden over an existing installation, your settings, API key, learned signals, whitelist, and blacklist are preserved. A safety backup is written to ~/MailWarden-upgrade-backup/ before anything changes, and is not deleted automatically so you can restore manually if needed.

System sleep

The filter and daily report only run while your Mac is awake. To keep MailWarden active around the clock, open System Settings → Battery → Options and turn on “Prevent automatic sleeping when the display is off”. Screen sleep is fine; full system sleep is not.
